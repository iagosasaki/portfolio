Projetos

Gerenciador de Férias

O Gerenciador de Férias é uma aplicação Full Stack desenvolvida com o objetivo de centralizar e organizar o planejamento da minha viagem para Salvador-BA, permitindo o cadastro estruturado de itinerários onde cada data pode possuir múltiplos horários e atividades relacionadas. No backend, a solução utiliza Node.js com o framework Fastify e TypeScript para garantir alta performance e segurança de tipos, empregando o Prisma ORM para gerenciar o banco de dados SQLite através de relações de um para muitos. A arquitetura da API foi desenhada de forma modular utilizando o sistema de plugins do Fastify e validação de esquemas com a biblioteca Zod, assegurando a integridade dos dados trafegados via JSON. O frontend foi construído com HTML5, CSS3 e JavaScript.

Asana Batch

O Asana Batch é uma solução de automação e Business Intelligence desenvolvida em Python com o objetivo de transformar dados brutos da plataforma Asana em insights visuais acionáveis, resolvendo a complexidade do acompanhamento manual de múltiplos projetos e tarefas. A ferramenta utiliza a biblioteca oficial asana para integração com a API, empregando python-dotenv para a gestão segura de credenciais e Pandas para a manipulação e agregação de grandes volumes de dados. A abordagem técnica destaca-se pelo processamento analítico de produtividade, utilizando as bibliotecas Matplotlib e NumPy para construir dashboards customizados que incluem métricas de conclusão por aluno, distribuição de status e rankings de performance.

BlocklyControl

Esta plataforma de ensino de robótica permite o controle de um braço robótico de quatro eixos através de uma interface de programação visual baseada na biblioteca Blockly. O projeto utiliza HTML5, CSS3 e JavaScript no frontend para traduzir blocos lógicos em comandos de hardware, comunicando-se com um backend em Python desenvolvido com FastAPI e Pydantic. O controle físico é executado por um microcontrolador ESP32, que utiliza WiFi.h e ESPAsyncWebServer para receber instruções via rede sem fio e gerenciar os servomotores através do driver PCA9685 com a biblioteca Adafruit. A solução oferece uma abordagem prática e acessível para iniciantes e crianças, unindo o desenvolvimento de software moderno à eletrônica para facilitar o aprendizado de estruturas de programação e automação.

PONG

Este projeto consiste em uma implementação de baixo nível do clássico jogo Pong para a plataforma Arduino, desenvolvida para demonstrar a geração de sinal de vídeo analógico e processamento em tempo real em hardware de recursos limitados. A aplicação utiliza a linguagem C++ e a biblioteca TVout para gerenciar a saída de vídeo NTSC, realizando a renderização de gráficos e fontes diretamente na tela. A abordagem técnica destaca-se pela manipulação direta de registradores para controle de entradas e o uso de potenciômetros via conversores analógico-digitais para a movimentação das raquetes. Estruturado sobre uma máquina de estados finitos, o sistema gerencia modos de menu, PvP e treino, implementando uma lógica de física para detecção de colisões e rebatidas dinâmicas.
